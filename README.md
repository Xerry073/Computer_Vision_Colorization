# Computer_Vision_Colorization
This is a Project of Course Computer Vision


1、任务说明：

图像上色是一个典型的条件生成任务。模型输入灰度图（L通道），预测其对应的色彩信息（ab通道）。需掌握 Lab 色彩空间转换，并理解生成模型如何为单一亮度信息分配合理的色彩语义。


2、数据集：

● ImageNet-Mini 或 CIFAR-10：利用彩色数据集，将其转化为灰度图作为输入，原图作为标签。

● 链接：Kaggle: ImageNet-1000

评测指标：

● 主观色彩合理性：物体（如草地、天空、人脸）的颜色是否符合现实逻辑。

● 色调分布对比：预测色彩分布与真实色彩分布的直方图相似度。

数据流
详细内容与维度

输入 (Input)

灰度亮度通道 (L Channel)：Lab 色彩空间中的亮度分量，维度为 [H, W, 1]。

数值范围通常为 [0, 100] 或归一化至 [-1, 1]。

输出 (Output)

1. 色彩通道预测 (ab Channels)：预测的 a（红绿）和 b（蓝黄）分量，维度为 [H, W, 2]。

2. 全彩图像：将输入 L 通道与预测 ab 通道拼接后转换回的 RGB 图像，维度为 [H, W, 3]。
